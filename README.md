# ☕ Java Programming Tasks

A collection of Java programs demonstrating fundamental programming concepts, arrays, searching, sorting, recursion, and problem-solving techniques.

## 📌 Programs Included

| File Name | Description |
|-----------|-------------|
| GFG.java | Print elements of an array using a `for` loop |
| BinarySearch.java | Binary Search implementation |
| Iterative Binary Search.java | Iterative Binary Search algorithm |
| MaxElementFinder.java | Find the maximum element in an array |
| KthSmallestSorting.java | Find the kth smallest element using sorting |
| ArrayPairs.java | Print all possible pairs of array elements |
| FibonacciLambda.java | Generate Fibonacci numbers using Java Streams and Lambda expressions |
| Palindrome.java | Check whether a number is a palindrome |
| LastDigits.java | Calculate the sum of the last digits of two numbers |
| Fruits.java | Java program demonstrating basic string/array operations with fruits |

## 🚀 Technologies Used

- Java
- JDK 17+
- VS Code
- Git & GitHub

## ▶️ How to Run

### Compile

```bash
javac FileName.java
```

Example:

```bash
javac BinarySearch.java
```

### Run

```bash
java FileName
```

Example:

```bash
java BinarySearch
```
